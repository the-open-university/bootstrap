Colours

Background colours

Colour utility classes allow you to modify an elements background or text colour by setting the relevant classes on the element to be styled.

The following classes can be applied to elements in order to change the background colour of the element:

Darker Background Colours

{% highlight html %} .bg-black .bg-dark-grey .bg-red .bg-pink .bg-dark-blue .bg-purple .bg-orange .bg-burgundy .bg-dark-green .bg-blue .bg-green .bg-teal {% endhighlight %}

Lighter Background Colours

{% highlight html %} .bg-white .bg-grey .bg-light-green .bg-turquoise .bg-light-blue .bg-yellow .bg-beige .bg-light-grey {% endhighlight %}
