Creatives

A creative allows a 100% width "showpiece" in the header area where you can place a large image, video, hero, carousel etc. You can see an example of a creative in action at the top of this page.

The creative will only take up the full width of the page if placed outside of a .container element. If the creative is inside a .container it reverts to a hero element, as shown below.

You can have an optional text overlay, seen above, or leave it out and just feature a decorative image.
